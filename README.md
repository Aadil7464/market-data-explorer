# Market Data Explorer

## Tech Stack

### Frontend
- React
- Material UI
- AG Grid
- Axios

### Backend
- Node.js
- Express.js
- Express-FileUpload

### Python Service
- FastAPI
- Pandas
- PyArrow


## Installation

### Frontend

```bash
npm run start
```

### Backend

```bash
npm run backend
```

### Python Service

```bash
npm run python-server
```
