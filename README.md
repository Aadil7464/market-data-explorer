# Market Data Explorer

## Tech Stack

### Frontend
- React
- Material UI
- AG Grid
- Axios

### Backend
- Node.js
- Express.js
- Express-FileUpload

### Python Service
- FastAPI
- Pandas
- PyArrow

### DevOps
- Docker
- Docker Compose

## Installation

### Frontend

```bash
cd client
npm install
npm start
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Python Service

```bash
cd python-service
pip install -r requirements.txt
uvicorn app:app --reload
```

### Docker

```bash
docker-compose up --build
